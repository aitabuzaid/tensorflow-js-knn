Train a Machine Learning model with TensorFlow.JS and a Webcam!

Try locating different objects in front of the webcam, for example, a face, a cat, a pencil, etc. Whenever an object is in view, click on one of the buttons to add it to a class. Take multiple images of each class, by rotation or translation, to improve the model accuracy. Once you are satisfied with the training, use the model to predict the of each object.
